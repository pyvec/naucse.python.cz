Python místo kalkulačky

  1. Doma (nebo pár dní po instalaci, pokud nejsi na srazu) si znovu pusť Python a zkus jestli stále funguje. 
    >>> 1 + 1
2
  2. Funguje i odečítání? 
    >>> 4 - 2
  3. A co násobení? Programátoři nenásobí pomocí · ani ×, použijí jiný operátor: hvězdičku. Vyzkoušej si to.
  4. Dělení? Znak ÷ se taky na klávesnici těžko píše (zvlášť na české). Jak se asi bude dělit?
  5. Závorky v Pythonu fungují jako v matematice. Zkus pomocí Pythonu vypočítat: 3 + (4 + 6) × 8 ÷ 2 - 1 = Jak se to zapíše v Pythonu?
  6. Jsou i jiné operátory než +, - a ty pro násobení a dělení.
    Co dělá s čísly operátor % (procento)?
  7. A co dělá operátor ** (dvě hvězdičky)?
  8. Až budeš příště sahat po kalkulačce, použij místo ní Python.

Hrátky s řetězci

  1. Python umí i jiné věci než čísla. Třeba takové řetězce – slova, věty nebo jiný text. Řetězce se zadávají v uvozovkách – jednoduchých ' nebo dvojitých ": 
    >>> "Ahoj!"
"Ahoj!"
  2. Řetězce jdou spojovat sčítáním. Vyzkoušej si to: 
    >>> "A" + "B"
"AB"
  3. Co je tady špatně? Jak to spravit? 
    >>> "Ahoj" + "PyLadies!"
  4. Řetězce se dají sčítat. Dají se i násobit? Dělit? Odečítat?
  5. Co se stane, když se pokusím sečíst číslo a řetězec?
